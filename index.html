F1 Dashboard

This website is an F1 Dashboard that allows users to explore and view details of Formula 1 races by selecting a season. On the homepage, users can select a season from a dropdown menu and view the races for that year. Upon selection, the website displays a list of races in a table, including their rounds, race names, and a button to view detailed results. Users can click on "Results" for each race to access in-depth qualifying and race results, including drivers, constructors, and performance times. The site features a functional navigation system where users can return to the homepage by clicking on the logo in the top left corner. With clean layouts and an engaging design, this dashboard delivers an intuitive and seamless experience for Formula 1 fans.
